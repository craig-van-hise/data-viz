
### 🤖 AGENT TASK: Implement Absolute Bounding Math for Tooltip Collisions

**Target File:** `composers.html`

**Objective:** Fix a critical UI bug where tooltips are clipping the top of the viewport. The previous collision logic failed to account for the sticky header obscuring the top of the window, resulting in inaccurate `spaceAbove` calculations.

**Implementation Steps:**

1. Locate the `marker.addEventListener('mouseenter', function () { ... })` block inside the `createRows()` function.
2. Replace the entire contents of that event listener with the following strict mathematical bounds calculation:

```javascript
const rect = this.getBoundingClientRect();
const tooltip = this.querySelector('.film-tooltip');

// 1. Get the exact bottom boundary of the sticky header
const stickyHeader = document.querySelector('.timeline-header');
// Add 15px extra padding to the header's bottom to be safe
const headerBottom = stickyHeader ? stickyHeader.getBoundingClientRect().bottom + 15 : 0;

// 2. Calculate true usable space above and below
const trueSpaceAbove = rect.top - headerBottom;
const trueSpaceBelow = window.innerHeight - rect.bottom;

// 3. The Math: If we have less than 350px above (safe buffer for tall posters) 
// AND there is more room below than above, flip it down.
if (trueSpaceAbove < 350 && trueSpaceBelow > trueSpaceAbove) {
    tooltip.classList.add('tooltip-down');
} else {
    tooltip.classList.remove('tooltip-down');
}
```

**Validation:** Ensure no other logic inside the `mouseenter` listener is preserved; this block completely replaces the previous positioning logic.
