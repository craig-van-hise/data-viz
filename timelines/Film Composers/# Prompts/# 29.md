
### 🤖 AGENT TASK: Script Purge & Blacklist Implementation

**Context for Agent:** The `scripts/` directory is cluttered with deprecated versions, debug files, and obsolete scraping logic. We need to isolate the core pipeline scripts and move the rest into an archive. Additionally, we need to implement a film "blacklist" so the user can easily exclude unwanted TMDb entries (like documentaries or shorts) from the timeline without them reappearing on the next data sync.

**Your Task:**

**1. The Great Script Purge (Archiving)**
Do not delete files permanently. Instead, create a new directory: `scripts/archive/`.
Move **everything** in the `scripts/` folder into `scripts/archive/` **EXCEPT** for the following core maintenance scripts:
* `fetch_tmdb_filmographies.py`
* `patch_tmdb_filmographies.py`
* `scrape_and_verify_links.py`
* `update_html.py`
* `scrape_thumbnails.py` (if it exists and is not deprecated)
* `fetch_composer_dates.py` (keep the most finalized version, archive the `v2`, `v3`, `threaded`, etc.)
* `final_validation.py`

**2. Create the Blacklist Data Structure**
Create a new file at `data/blacklist.json`. Initialize it with a structure that maps a composer's name to an array of exact film titles to exclude.
```json
{
  "Composer Name": [
    "Name of Junk Documentary",
    "Name of Unwanted Short Film"
  ]
}
```

**3. Implement Blacklist Logic in TMDb Scripts**
Modify **both** `scripts/fetch_tmdb_filmographies.py` and `scripts/patch_tmdb_filmographies.py` to respect this new blacklist.
* **Load the Blacklist:** Near the top of the script (where `COMPOSERS_LIST_PATH` is loaded), add logic to safely load `data/blacklist.json`. If the file doesn't exist or is empty, default to an empty dictionary `{}`.
* **Apply the Filter (Step C / D):** During the loop where films are formatted and added to the list, check if the current `composer` exists in the blacklist dictionary. If they do, check if the current `film.get("title")` is in their blacklist array.
* **Exclusion:** If the title is blacklisted, `continue` (skip adding it to `formatted_list`). Add a small print statement for logging: `print(f"  🚫 Skipped blacklisted film: {title}")`.

**4. Execution & Validation**
Run a quick dry-run of the script or verify the logic to ensure the blacklist correctly blocks a dummy entry without crashing. Summarize the files moved to the archive once complete.
