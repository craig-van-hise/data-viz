

# Agent Task: Decouple Hover Animation & Extend Track Pill

**Context:** We need to apply two visual tweaks to the timeline. First, the "Most Important Films" flags are scaling up when hovered; we need to isolate the hover animation so it *only* scales the visual dot (`.marker-dot`), leaving the flags static. Second, the leftmost dot is overflowing the left edge of the timeline track. We need to visually extend the track's background pill to encapsulate the dots with proper padding using a `::before` pseudo-element, ensuring we don't break the internal coordinate space math.

**File to Modify:** `composer_profile.html`

**Instructions:**

### 1. Decouple the Marker Styling & Hover Logic (CSS)
Locate the `.film-marker` and `.film-marker:hover` classes in the `<style>` block.
We need to strip the visual styling from the wrapper and apply it to the `.marker-dot` class, then update the hover selector.

**Replace the existing `.film-marker` and `.film-marker:hover` blocks with these three updated rules:**

```css
        .film-marker {
            position: absolute;
            top: 50%;
            width: 12px;
            height: 12px;
            transform: translate(-50%, -50%);
            cursor: pointer;
            z-index: 10;
            /* Visual styles moved to .marker-dot */
        }

        .marker-dot {
            width: 100%;
            height: 100%;
            border-radius: 50%;
            background-color: var(--marker-color);
            transition: all 0.3s cubic-bezier(0.25, 0.8, 0.25, 1);
            box-shadow: 0 0 10px rgba(138, 180, 248, 0.3);
            position: relative;
            z-index: 1; /* Keeps dot below the flags */
        }

        /* Target ONLY the dot on hover so flags remain completely static */
        .film-marker:hover .marker-dot {
            transform: scale(1.5);
            background-color: #fff;
            box-shadow: 0 0 20px rgba(138, 180, 248, 0.6);
        }
```

### 2. Visually Extend the Timeline Track (CSS)
Locate the `.timeline-track` class in the `<style>` block. 
We need to remove the background properties from the main container and apply them to an extended pseudo-element.

**Update `.timeline-track` and add a new `::before` rule:**

```css
        .timeline-track {
            position: relative;
            height: 60px;
            margin-top: 20px;
            /* Background and border-radius removed to be handled by ::before */
        }

        /* Pseudo-element extends the visual background 30px outward on both sides */
        .timeline-track::before {
            content: '';
            position: absolute;
            top: 0;
            left: -30px; 
            right: -30px; 
            bottom: 0;
            background: rgba(255, 255, 255, 0.02);
            border-radius: 30px;
            z-index: 0;
        }
```
