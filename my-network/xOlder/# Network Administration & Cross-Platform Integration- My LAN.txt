# Network Administration & Cross-Platform Integration: My LAN

## MAC-LAPTOP
* **Role:** Remote Management & Admin Console
* **Primary Storage:** 256GB (Internal)

## MAC-MINI
* **Role:** Network Hub & Agent Sandbox
* **Primary Storage:** * 256GB (Internal)
    * 1TB M.2 NVMe (Thunderbolt 4)
    * 2TB M.2 NVMe (Thunderbolt 4)
* **Centralized Storage Array:** 8-Bay Array, 32TB Total (USB 3)
    * *Function:* Mixture of Primary, Secondary, and Tertiary storage for MAC-MINI, MAC-LAPTOP, and WIN-LAPTOP.

## WIN-WORKSTATION
* **Role:** Primary Compute & Production
* **Primary Storage:** 8TB M.2 NVMe (Internal)
* **Secondary Storage:** 8TB SATA III SSD (Internal)

## WIN-LAPTOP
* **Role:** Mobile Workstation & High-Performance Node
* **Primary Storage:** 2TB SATA III SSD (Internal)

---

## Network Topology & Interconnects

* **Global Wireless LAN:** * **5GHz Wi-Fi** (Connects all nodes on the same local network)
* **Point-to-Point Links:**
    * **1Gb Gigabit Ethernet:** Direct hardline connection between `WIN-WORKSTATION` and `WIN-LAPTOP`.
    * **Thunderbolt 4 Bridge:** Direct high-speed connection between `MAC-MINI` and `WIN-WORKSTATION`. 
        * *Protocol:* Configured as a cross-platform network share utilizing **SMB (Server Message Block)**.
* **Remote Administration:**
    * `MAC-LAPTOP` acts as the central control console. It accesses and manages the other nodes on the network utilizing two separate screen-sharing applications: **Jump Desktop** and **Remote Desktop**.








## Let's make a network diagram:

#### Phase 1: Establish the Nodes (The Core Layout)
1. **Top Layer (Management):** Place a node at the top center of your canvas. Label it `MAC-LAPTOP`.
2. **Middle Layer (The High-Compute Core):** Place two nodes side-by-side in the center of the canvas, below the Mac Laptop. 
   * Label the left node `MAC-MINI`.
   * Label the right node `WIN-WORKSTATION`.
3. **Middle Layer (The Mobile Workstation):** Place a node to the right of the `WIN-WORKSTATION`. Label it `WIN-LAPTOP`.

#### Phase 2: Add Peripherals (The External I/O)
Draw the external storage units attached directly to their respective host machines.

1. **The NVMe Drives (Mac Mini):** Draw two smaller, identical boxes positioned slightly below and to the left of the `MAC-MINI`.
   * Label Box 1: `1TB M.2 NVMe (Primary)`
   * Label Box 2: `2TB M.2 NVMe (Primary)`
   * Draw a direct line from the `MAC-MINI` to *each* of these boxes. Label both of those lines **Thunderbolt 4**.
2. **The Unified Storage Hub (Mac Mini):** Draw a larger box below and to the right of the `MAC-MINI`.
   * Label this box: `8-Bay Array (all 4TB) (Mixture of Pri/Sec/Ter)`
   * Draw a direct line from the `MAC-MINI` to this large box. Label this line **USB 3**.
3. **The Workstation Archive (Windows Workstation):** Draw a rectangular box directly below the `WIN-WORKSTATION`.
   * Label this box: `4-Bay Array (all 2TB) (Tertiary Storage)`
   * Draw a direct line from the `WIN-WORKSTATION` to this box. Label this line **USB 3**.

#### Phase 3: Add Global Connectivity (The Wi-Fi LAN)
1. Draw a large, centered horizontal bar or a "Cloud" icon in the middle of the diagram, positioned between the top `MAC-LAPTOP` and the middle row of computers.
2. Label this core element `Global Network: 5GHz Wi-Fi LAN`.
3. Draw individual lines from **all four computers** directly to this central bar. *(This visually establishes that every device is on the same subnet).*

#### Phase 4: Add Point-to-Point Links (The High-Speed Hardlines)
Add the specific high-speed connections directly between the computer nodes.

1. **The Windows Hardline:** Draw a solid line directly connecting the `WIN-WORKSTATION` and the `WIN-LAPTOP`. Label this line `1Gb Gigabit Ethernet`.
2. **The Cross-Platform Hardline:** Draw a bold or distinctly colored dashed line directly connecting the `MAC-MINI` and the `WIN-WORKSTATION`. Label this line `Thunderbolt 4 Bridge (SMB)`.

#### Phase 5: Add Remote Administration (The Management Links)
Map out how the management laptop controls the network.

1. **To the Mac Mini:** Draw a line starting from the `MAC-LAPTOP` and connecting to the `MAC-MINI`. Label this line `Jump Desktop`.
2. **To the Windows Nodes:** Draw a line starting from the `MAC-LAPTOP` and pointing to the `WIN-WORKSTATION` and `WIN-LAPTOP` (you can bracket them or draw two separate lines). Label this line `Remote Desktop`.
