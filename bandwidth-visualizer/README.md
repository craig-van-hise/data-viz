# Digital Bandwidth Visualizer

A modern, interactive data visualization tool that compares the real-world vs. theoretical speeds of various digital interfaces (Internet, Network, Peripherals, Storage, System PCIe).

## Features
- **Interactive Chart:** Built with Recharts for responsive and smooth visualizations.
- **Dynamic Filtering:** Filter data by categories like "Internet", "Network", "Bluetooth", etc.
- **Scale Toggling:** Switch between Linear and Logarithmic scales to compare vast data ranges (from Dial-up to PCIe 6.0).
- **Comparison View:** Visually distinguishes between Real World and Theoretical performance.

## Tech Stack
- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS (implied via standard modern stack) or CSS Modules
- **Animation:** Framer Motion
- **Icons:** Lucide React
- **Charts:** Recharts

## Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run locally:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
